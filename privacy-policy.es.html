Política de Privacidad

Última actualización: junio de 2026

Introducción

FlightFinder es una aplicación de seguimiento de aeronaves que utiliza datos ADS-B públicos a través de OpenSky Network para mostrar aviones cerca de su ubicación. Esta política de privacidad explica qué información se recopila y cómo se maneja.

En resumen: FlightFinder no recopila datos personales, no utiliza SDK de análisis ni muestra publicidad. El único dato que sale de su dispositivo es su ubicación aproximada, enviada únicamente para consultar a OpenSky Network sobre aviones cercanos.

1. Información que recopilamos

1.1 Ubicación aproximada

FlightFinder utiliza la ubicación aproximada de su dispositivo (precisión a nivel de ciudad) para consultar la API de OpenSky Network en busca de aeronaves en su zona.

  • Su ubicación aproximada se envía a OpenSky Network únicamente para obtener datos públicos de transpondedores de aeronaves en su área.
  • No se almacena en su dispositivo más allá de la sesión actual.
  • No se comparte con ningún otro tercero.
  • No se asocia con ningún identificador personal como una cuenta o ID de dispositivo.

1.2 Acceso a la cámara

FlightFinder puede solicitar acceso a la cámara para la vista de realidad aumentada (RA) opcional. Cuando está habilitada:

  • El flujo de la cámara se procesa completamente en su dispositivo.
  • Ninguna imagen, video o dato de la cámara se transmite, almacena o comparte.
  • El acceso a la cámara se puede revocar en cualquier momento en la configuración del dispositivo.

1.3 Datos de sensores

FlightFinder utiliza el giroscopio, acelerómetro y magnetómetro de su dispositivo para determinar la orientación del dispositivo para la función de puntero celeste.

  • Todos los datos de sensores se procesan localmente en su dispositivo.
  • Nunca se transmiten ni almacenan.

1.4 Lo que NO recopilamos

FlightFinder no recopila, almacena ni transmite:

  • Nombres, direcciones de correo electrónico o información personal identificable.
  • Credenciales de cuenta o información de inicio de sesión.
  • Identificadores de dispositivo, IMEI o ID de publicidad.
  • Información de pago.
  • Historial de navegación, registros de fallos o análisis de uso.

2. Servicios de terceros

2.1 OpenSky Network

Cuando usa FlightFinder, su ubicación aproximada se envía a OpenSky Network para obtener datos públicos de transpondedores ADS-B de aeronaves en su área. OpenSky Network es una asociación sin fines de lucro que proporciona acceso abierto a datos ADS-B para investigación y uso no comercial.

2.2 ADSBDB

FlightFinder consulta la API de ADSBDB utilizando códigos públicos de transpondedor ICAO24 de aeronaves para enriquecer la información de vuelo. No se transmiten datos del usuario.

2.3 Airport-data.com

FlightFinder consulta airport-data.com para obtener fotografías de aeronaves utilizando matrículas públicas. No se transmiten datos del usuario.

2.4 Flagsapi.com

FlightFinder utiliza flagsapi.com para mostrar banderas de países basadas en códigos ISO de país derivados de la ruta del avión. No se transmiten datos del usuario.

3. Almacenamiento de datos

FlightFinder almacena solo las preferencias del usuario localmente en su dispositivo mediante EncryptedSharedPreferences de Android (cifrado AES-256). Estas preferencias incluyen:

  • Preferencias de visualización (altitud, logotipos de aerolíneas, estela de condensación)
  • Unidades de medida (metros/pies, kilómetros/millas, km/h/mph)
  • Selección de idioma
  • Ajustes de campo de visión
  • Radio de búsqueda
  • Número de vuelos vistos (para la solicitud de calificación)

No se almacenan datos de vuelo, ubicación o datos personales de forma persistente. Todas las preferencias se eliminan al desinstalar la aplicación.

4. Privacidad infantil

FlightFinder no recopila a sabiendas información personal de niños menores de 13 años. Si es padre o tutor y cree que su hijo ha proporcionado datos personales, contáctenos para que podamos investigar y eliminar dichos datos.

5. Cambios a esta política

Podemos actualizar esta política de privacidad ocasionalmente. Los cambios se reflejarán en esta página con una fecha de revisión actualizada. Le recomendamos revisar esta página periódicamente.
