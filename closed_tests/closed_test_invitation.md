# Flight Finder : Recherche Beta testeurs pour Android

Je viens vers vous aujourd'hui car j'ai développé une application Android et  j'ai besoins de Beta Testeurs pour mener une camapgne de test avant publication officielle sur le Play Store.

**Si comme moi vous regardez souvent les avions passer dans le ciel en vous évadant, en imaginant quelle est leur destination cette application est faite pour vous**

# Pointez votre téléphone vers le ciel et identifier les avions

Avec **FlightFinder**, vous pouvez pointer les avions dans le ciel et les afficher sur une vue en réalité augmentée, visualiser les modèles d'avions, leur compagnie et de nombreuses autres données. FlightFinder est une application complétement gratuite.

J'ai besoin de vos retours !!!

# Comment aider à publier l'application

L'application est en **closed testing** pendant **14 jours** (obligation Google Play). Pour me soutenir :

1. **Rejoignez le groupe Google** FlightFinder Beta : [`https://groups.google.com/g/flight-finder-beta`](https://groups.google.com/g/flight-finder-beta)
2. **Téléchargez l'application** sur le Play Store:  [https://play.google.com/store/apps/details?id=com.lewisnclarkexploration.flightfinder](https://play.google.com/store/apps/details?id=com.lewisnclarkexploration.flightfinder)
3. **Laissez l'application installée** pendant les 14 jours de test.

Pas besoin de tester activement tous les jours ? juste l'installer et l'utiliser quand vous voulez pour identifier les avions dans le ciel.

*FlightFinder est gratuit. Construit avec passion et du ciel plein les yeux pour les reveurs.*
