개인정보처리방침

최종 업데이트: 2026년 6월

소개

FlightFinder는 OpenSky Network를 통해 공개 ADS-B 데이터를 사용하여 사용자 주변의 항공기를 표시하는 항공기 추적 애플리케이션입니다. 본 방침은 수집되는 정보와 그 처리 방법을 설명합니다.

요약: FlightFinder는 개인 데이터를 수집하지 않으며, 분석 SDK를 사용하지 않고, 광고를 표시하지 않습니다. 기기에서 전송되는 유일한 데이터는 사용자의 대략적인 위치이며, 이는 OpenSky Network에 주변 항공기를 조회하기 위한 목적으로만 사용됩니다.

1. 수집하는 정보

1.1 대략적인 위치

FlightFinder는 기기의 대략적인 위치(도시 수준 정확도)를 사용하여 OpenSky Network API에 주변 항공기를 조회합니다.

  • 대략적인 위치는 사용자 지역의 공개 항공기 트랜스폰더 데이터를 가져오기 위한 목적으로만 OpenSky Network에 전송됩니다.
  • 현재 세션 이후에는 기기에 저장되지 않습니다.
  • 제3자와 공유되지 않습니다.
  • 계정이나 기기 ID와 같은 개인 식별자와 연결되지 않습니다.

1.2 카메라 접근

FlightFinder는 선택적 증강 현실(AR) 보기를 위해 카메라 접근을 요청할 수 있습니다. 활성화된 경우:

  • 카메라 스트림은 전적으로 사용자 기기에서 처리됩니다.
  • 카메라의 이미지, 비디오 또는 데이터는 전송, 저장 또는 공유되지 않습니다.
  • 카메라 접근 권한은 기기 설정에서 언제든지 철회할 수 있습니다.

1.3 센서 데이터

FlightFinder는 SkyPointer 기능의 방향을 결정하기 위해 기기의 자이로스코프, 가속도계 및 자기계를 사용합니다.

  • 모든 센서 데이터는 사용자 기기에서 로컬로 처리됩니다.
  • 절대 전송 또는 저장되지 않습니다.

1.4 수집하지 않는 정보

FlightFinder는 다음을 수집, 저장 또는 전송하지 않습니다:

  • 이름, 이메일 주소 또는 개인 식별 정보.
  • 계정 자격 증명 또는 로그인 정보.
  • 기기 식별자, IMEI 또는 광고 ID.
  • 결제 정보.
  • 검색 기록, 충돌 로그 또는 사용 분석 데이터.

2. 제3자 서비스

2.1 OpenSky Network

FlightFinder를 사용할 때 사용자의 대략적인 위치는 OpenSky Network로 전송되어 사용자 지역의 공개 ADS-B 트랜스폰더 데이터를 가져옵니다. OpenSky Network는 연구 및 비상업적 용도를 위한 ADS-B 데이터의 개방형 액세스를 제공하는 비영리 단체입니다.

2.2 ADSBDB

FlightFinder는 항공기의 공개 ICAO24 트랜스폰더 코드를 사용하여 ADSBDB API에 조회하여 항공편 정보를 보강합니다. 사용자 데이터는 전송되지 않습니다.

2.3 Airport-data.com

FlightFinder는 항공기 사진을 가져오기 위해 airport-data.com에 조회합니다. 사용자 데이터는 전송되지 않습니다.

2.4 Flagsapi.com

FlightFinder는 항공편 경로에서 파생된 ISO 국가 코드를 기반으로 국기를 표시하기 위해 flagsapi.com을 사용합니다. 사용자 데이터는 전송되지 않습니다.

3. 데이터 저장

FlightFinder는 Android의 EncryptedSharedPreferences(AES-256 암호화)를 사용하여 사용자 기본 설정만 기기에 로컬로 저장합니다. 이러한 기본 설정에는 다음이 포함됩니다:

  • 표시 설정(고도, 항공사 로고, 비행운)
  • 측정 단위(미터/피트, 킬로미터/마일, km/h/mph)
  • 언어 선택
  • 시야 설정
  • 검색 반경
  • 조회된 항공편 수(평가 요청용)

어떤 항공편 데이터, 위치 데이터 또는 개인 데이터도 영구적으로 저장되지 않습니다. 앱을 제거하면 모든 기본 설정이 삭제됩니다.

4. 아동 프라이버시

FlightFinder는 13세 미만의 어린이로부터 의도적으로 개인 정보를 수집하지 않습니다. 귀하가 부모 또는 보호자이며 자녀가 개인 데이터를 제공했다고 생각되는 경우, 당사에 연락해 주시면 해당 데이터를 조사하고 삭제하겠습니다.
