Политика конфиденциальности

Последнее обновление: июнь 2026 г.

Введение

FlightFinder — это приложение для отслеживания самолетов, которое использует общедоступные данные ADS-B через OpenSky Network для отображения воздушных судов рядом с вами. Настоящая политика конфиденциальности объясняет, какая информация собирается и как она обрабатывается.

Кратко: FlightFinder не собирает личные данные, не использует SDK аналитики и не показывает рекламу. Единственные данные, покидающие ваше устройство — это ваш приблизительный местоположение, отправляемое исключительно для запроса к OpenSky Network о ближайших самолетах.

1. Собираемая информация

1.1 Приблизительное местоположение

FlightFinder использует приблизительное местоположение вашего устройства (точность на уровне города) для запроса API OpenSky Network о воздушных судах в вашем районе.

  • Ваше приблизительное местоположение отправляется в OpenSky Network исключительно для получения общедоступных данных транспондеров самолетов в вашем районе.
  • Не сохраняется на вашем устройстве после завершения сессии.
  • Не передается третьим лицам.
  • Не связывается с личными идентификаторами, такими как учетная запись или ID устройства.

1.2 Доступ к камере

FlightFinder может запрашивать доступ к камере для дополнительного режима дополненной реальности (AR). При включении:

  • Поток с камеры обрабатывается полностью на вашем устройстве.
  • Никакие изображения, видео или данные с камеры не передаются, не сохраняются и не распространяются.
  • Доступ к камере можно отозвать в любое время в настройках устройства.

1.3 Данные датчиков

FlightFinder использует гироскоп, акселерометр и магнитометр вашего устройства для определения ориентации для функции небесного указателя.

  • Все данные датчиков обрабатываются локально на вашем устройстве.
  • Никогда не передаются и не сохраняются.

1.4 Что мы НЕ собираем

FlightFinder не собирает, не хранит и не передает:

  • Имена, адреса электронной почты или личную информацию.
  • Учетные данные или информацию для входа.
  • Идентификаторы устройства, IMEI или рекламные ID.
  • Платежную информацию.
  • Историю просмотров, журналы сбоев или аналитику использования.

2. Сторонние сервисы

2.1 OpenSky Network

При использовании FlightFinder ваше приблизительное местоположение отправляется в OpenSky Network для получения общедоступных данных транспондеров ADS-B самолетов в вашем районе. OpenSky Network — это некоммерческая ассоциация, предоставляющая открытый доступ к данным ADS-B для исследований и некоммерческого использования.

2.2 ADSBDB

FlightFinder запрашивает API ADSBDB, используя общедоступные коды транспондеров ICAO24 самолетов для обогащения информации о рейсе. Данные пользователя не передаются.

2.3 Airport-data.com

FlightFinder запрашивает airport-data.com для получения фотографий самолетов по их государственным регистрационным номерам. Данные пользователя не передаются.

2.4 Flagsapi.com

FlightFinder использует flagsapi.com для отображения флагов стран на основе кодов ISO стран, полученных из маршрута полета. Данные пользователя не передаются.

3. Хранение данных

FlightFinder хранит только пользовательские настройки локально на вашем устройстве с помощью EncryptedSharedPreferences Android (шифрование AES-256). Эти настройки включают:

  • Настройки отображения (высота, логотипы авиакомпаний, инверсионный след)
  • Единицы измерения (метры/футы, километры/мили, км/ч/миль/ч)
  • Выбор языка
  • Настройки поля зрения
  • Радиус поиска
  • Количество просмотренных рейсов (для запроса оценки)

Никакие данные о полетах, местоположении или личные данные постоянно не сохраняются. Все настройки удаляются при удалении приложения.

4. Конфиденциальность детей

FlightFinder намеренно не собирает личную информацию от детей младше 13 лет. Если вы родитель или опекун и считаете, что ваш ребенок предоставил личные данные, свяжитесь с нами для расследования и удаления таких данных.

5. Изменения политики

Мы можем время от времени обновлять эту политику конфиденциальности. Изменения будут отражены на этой странице с обновленной датой пересмотра. Рекомендуем периодически просматривать эту страницу.
