主要功能

📡

实时天空指针

举起手机，在真实天空上看到叠加的飞机标签。陀螺仪 + 指南针 + AR 相机视图，即时识别。

🌍

世界地图

在地图上实时追踪您所在区域的所有航班。点击任何飞机查看其路线、航空公司和目的地。

✈️

航班详情

丰富的航班卡片，包含照片、可信度评分、路线可视化和国家追踪——全部实时更新。

🗺️

国家探索

每次发现前往新国家的航班时都可以庆祝。交互式热力图显示您的全球覆盖范围。

隐私优先

FlightFinder 被设计为默认保护隐私。无账户、无分析 SDK、无个人数据收集。您的近似位置仅发送给 OpenSky Network 以获取附近飞机信息——不会从您的设备传出任何其他信息。

阅读完整隐私政策 →

工作原理

FlightFinder 结合来自 OpenSky Network 的公共 ADS-B 数据和来自 ADSBDB 的补充信息，提供全面的航班信息，无需任何 API 密钥或用户账户。飞机位置每 20 秒更新一次，并平滑插值以实现流畅的实时追踪。
