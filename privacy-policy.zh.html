隐私政策

最后更新：2026年6月

介绍

FlightFinder 是一款航空追踪应用，通过 OpenSky Network 使用公开的 ADS-B 数据来显示您附近的飞机。本隐私政策说明收集了哪些信息以及如何处理这些信息。

简而言之：FlightFinder 不收集个人数据，不使用分析 SDK，也不显示广告。唯一离开您设备的数据是您的近似位置，仅用于向 OpenSky Network 查询附近的飞机。

1. 我们收集的信息

1.1 近似位置

FlightFinder 使用您设备的近似位置（城市级精度）来查询 OpenSky Network API 获取您所在区域的飞机信息。

  • 您的近似位置仅发送给 OpenSky Network，用于获取您所在区域的公开飞机应答器数据。
  • 不会在您的设备上存储超过当前会话。
  • 不会与任何第三方共享。
  • 不会与任何个人标识符（如账户或设备 ID）关联。

1.2 相机权限

FlightFinder 可能会请求相机权限以使用可选的增强现实（AR）视图。启用时：

  • 相机画面完全在您的设备上处理。
  • 不会传输、存储或共享相机的任何图像、视频或数据。
  • 您可以随时在设备设置中撤销相机权限。

1.3 传感器数据

FlightFinder 使用您设备的陀螺仪、加速度计和磁力计来确定设备方向，以实现天空指针功能。

  • 所有传感器数据均在您的设备上本地处理。
  • 绝不会传输或存储。

1.4 我们不收集的信息

FlightFinder 不收集、存储或传输：

  • 姓名、电子邮件地址或任何个人身份信息。
  • 账户凭据或登录信息。
  • 设备标识符、IMEI 或广告 ID。
  • 支付信息。
  • 浏览历史、崩溃日志或使用分析。

2. 第三方服务

2.1 OpenSky Network

当您使用 FlightFinder 时，您的近似位置会发送给 OpenSky Network，以获取您所在区域的公开 ADS-B 应答器数据。OpenSky Network 是一个非营利组织，为研究和非商业用途提供 ADS-B 数据的开放访问。

2.2 ADSBDB

FlightFinder 使用飞机的公开 ICAO24 应答器代码查询 ADSBDB API 以丰富航班信息。不会传输任何用户数据。

2.3 Airport-data.com

FlightFinder 查询 airport-data.com 以获取飞机的照片。不会传输任何用户数据。

2.4 Flagsapi.com

FlightFinder 使用 flagsapi.com 根据航班路线中的 ISO 国家代码显示国旗。不会传输任何用户数据。

3. 数据存储

FlightFinder 仅使用 Android 的 EncryptedSharedPreferences（AES-256 加密）在您的设备本地存储用户偏好设置。这些偏好设置包括：

  • 显示偏好（高度、航空公司标志、凝结尾迹）
  • 测量单位（米/英尺、公里/英里、公里/小时/英里/小时）
  • 语言选择
  • 视野设置
  • 搜索半径
  • 已查看航班数（用于评分提示）

不会持久存储任何航班数据、位置数据或个人数据。卸载应用时，所有偏好设置将被删除。

4. 儿童隐私

FlightFinder 不会故意收集 13 岁以下儿童的任何个人信息。如果您是家长或监护人，并且认为您的孩子提供了个人数据，请联系我们以便我们调查并删除这些数据。

5. 本政策的变更

我们可能不时更新本隐私政策。任何变更将在此页面上反映，并附有更新的修订日期。建议您定期查看此页面。
